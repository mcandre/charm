# charm - a minimal ncurses copy

## REQUIREMENTS

### Ubuntu

 - libgc-dev

### Mac OS X, Windows

 - [Boehm GC](http://www.hpl.hp.com/personal/Hans_Boehm/gc/)

	$ make
	$ ./hellocharm

	q
